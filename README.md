Motorbike Maintenance Tracker

Overview

A Flask + SQLite web app to track motorbike maintenance.

Features

Bike CRUD

Maintenance tracking

Due/overdue alerts

Filtering

Setup

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m flask --app app run --debug

Run

http://127.0.0.1:5000
